# Palindrome Number Checker (C)

## 📌 Project Description
This project checks whether a given number is a palindrome or not using C programming.

A palindrome number checker reads the same forward and backward.

---

## 🛠️ Technologies Used
- C Programming Language
- GCC Compiler
- VS Code

---

## 🧠 Algorithm
1. Take input from user
2. Store original number
3. Reverse the number using while loop
4. Compare original and reversed numbers
5. If equal → Palindrome
6. Else → Not Palindrome
